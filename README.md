# Practicum_projects
Автоматизация процесса обработки онлайн-заказов интернет-магазина «Переделкино»
Проект выполнен в рамках дипломной работы. Цель — автоматизация процесса обработки онлайн-заказов интернет-магазина стройматериалов и товаров для дома «Переделкино».

Магазин имеет действующий сайт с каталогом товаров и возможностью оформления заказа, но обработка заказов ведется полностью вручную через почтовый ящик, телефонные звонки на склад и клиентам, а также папки «Досогласовать» и «На отгруз». Это приводит к постоплатным отменам, потере заказов и высокой нагрузке на администратора.

Результатом проекта стала модель целевого процесса, архитектура решения и полный пакет документов для внедрения.

 Ключевые проблемы текущего процесса (AS IS)

- Отсутствие онлайн-остатков — клиент оплачивает товар, которого может не быть на складе
- Полностью ручная обработка заказов через почту и телефонные звонки
- Папка «Досогласовать» — прямые потери заказов из-за недозвона
- Отсутствие отслеживания статуса заказа для клиента
- Нет Личного кабинета и истории заказов
- Неудобная корзина: нельзя удалить отдельный товар
- Нет поиска по каталогу и поля для промокода

Предлагаемое решение (TO BE)

 Ключевые изменения

1. **Модуль онлайн-остатков** — резервирование товара на 30 минут при переходе в корзину, отображение актуального наличия
2. **Административная панель** — замена почтового ящика, автоматическая проверка наличия, управление заказами в один клик
3. **Автоматические уведомления** — СМС и Email на каждом этапе вместо ручных звонков
4. **Личный кабинет клиента** — история заказов, отслеживание статуса, повтор заказа в 1 клик
5. **Доработка интерфейса** — поиск, удаление отдельных позиций, промокоды

Бизнес-требования (SMART)

| ID | Требование | Целевой показатель | Срок |
|:---|:---|:---|:---|
| BR-01 | Увеличить конверсию корзина → заказ | +25% | 3 месяца |
| BR-02 | Исключить постоплатные отмены | 0% | 3 месяца |
| BR-03 | Сократить ручную работу администратора | -70% | 2 месяца |
| BR-04 | Исключить недозвоны как причину отмены | 0 случаев | 1 месяц |
| BR-05 | Увеличить долю повторных покупок | до 25% | 6 месяцев |
| BR-06 | Снизить количество брошенных корзин | -30% | 3 месяца |
